# 🗄️ SQL Fundamentals

A repository documenting my journey of learning and applying SQL through practical projects and exercises.

---

## 📚 About This Repository

This repo contains SQL scripts, mini-projects, and practice “homework” tasks completed while building a strong foundation in SQL and relational databases.

The focus is on understanding how databases work and writing efficient, readable queries.

---

## 🧠 Topics Covered

* Basic SELECT statements
* Filtering with WHERE
* Sorting with ORDER BY
* Aggregate functions (COUNT, SUM, AVG, MIN, MAX)
* GROUP BY & HAVING
* JOINS (INNER, LEFT, RIGHT)

More advanced topics will be added as I progress.
